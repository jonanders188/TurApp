# Tekster til første appskjermer

## Hjem
- Søk: Hvor vil du gå tur?
- Værkort: Perfekt turvær de neste 3 timene
- Seksjon: Turer nær deg
- Hurtigfiltre: Barnevogn, Bæremeis, Lett å gå, Rullestol, Barn, Under 2 t

## Filter
- Tittel: Filtrer turer
- Undertittel: Tilpass turtipsene etter dine behov
- Seksjon: Tilgjengelighet og egnethet
- CTA: Vis turer
- Sekundær: Nullstill

## Turdetalj
- CTA: Start tur
- Sekundær: Lagre tur
- Seksjon: Praktisk informasjon
- Infofelter: Underlag, Parkering, Toalett, Hunder

## Eksempel pitch
Finn turer i Vestfold som faktisk passer deg - enten du har barnevogn, bæremeis, hund, rullestol eller bare vil ha en enkel tur.
